# Music Control NeoForge

Unofficial NeoForge 1.21.1 port of Music Control.

This project is a community port of the original Fabric mod **Music Control** by Pardys / sf-inc.

## Original project

- Original mod: Music Control
- Original source: https://github.com/sf-inc/music_control
- Original Modrinth page: https://modrinth.com/mod/music-control
- Original license: MIT

## Status

This port is currently work in progress.

Implemented:

- NeoForge 1.21.1 project setup
- Client-only mod entrypoint
- Basic Music Control key mappings
- GitHub Actions build

Not implemented yet:

- Music playback control logic
- Config system
- GUI music panel
- Mixins
- Resource pack/custom music management

## Disclaimer

This is not an official release by the original author.
